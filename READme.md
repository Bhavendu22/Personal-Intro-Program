# Personal Introduction Program

## 1. Project Overview
This project is a user-friendly Python script designed to create a personalized welcome experience. The program interacts with the user by collecting basic personal details (Name, Age, Hobby, Location) and processing them to display a formatted, enthusiastic welcome message.

**Objectives:**
- To demonstrate basic Input/Output operations in Python.
- To utilize variables and data storage.
- To implement f-strings for dynamic text formatting.

## 2. Setup Instructions
Follow these steps to run the project on your local machine:

1. **Prerequisites**: Ensure you have Python 3.x installed. You can check this by typing `python --version` in your terminal.
2. **Download**: Clone this repository or download the `personal_intro.py` file.
3. **Execution**: Open your terminal or command prompt, navigate to the folder, and run:
   ```bash
   python personal_intro.py
